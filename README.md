# Python Fundamentals

This repository contains Python exercises and mini projects created during my learning journey in programming and software development.

## Topics covered
- Conditionals
- Loops
- Tuples
- Logic and problem solving

## Repository structure
- `conditionals/` → exercises focused on decision-making with `if`, `elif`, and `else`
- `loops/` → exercises using `for`, `while`, and repetition logic
- `tuples/` → exercises focused on tuple manipulation and analysis
- `logic-training/` → mini projects and logic-based programs combining multiple concepts

## Purpose
The goal of this repository is to document my progress while building a strong foundation in Python before moving to backend development and more advanced software engineering topics.
